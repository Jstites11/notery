# notery
Notery allows you to easily write and share notes with collaborators. 
Think of it as a mix of GroupMe and Google Docs.

See [devpost](https://devpost.com/software/notery]) for more info.

## How to Install
### Client
```
cd client
npm install
```
### Server
```
cd server
npm install
```
